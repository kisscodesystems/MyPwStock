Please download the source and compile it or download the class from
https://github.com/kisscodesystems/MyPwStock/blob/master/MyPwStock.jar
Please move MyPwStock.jar into a safe folder.
Open a command line and navigate into that folder.
Type "java -version" to see the actual version of your java. (min 1.8 is required.)
See that the "java" command points to the correct java executable!
Type "java -jar MyPwStock.jar ?" for the usable commands.
Type "java -jar MyPwStock.jar help" for more detailed information.
Type "java -jar MyPwStock.jar application story" if you want to read.
Type "java -jar MyPwStock.jar application describe" if you would like to get more details.
